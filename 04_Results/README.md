# 📈 Results & Policy Output (`/04_Results`)

## 📌 Overview
This directory contains the final synthesis outputs of the Dhaka PM₂.₅ forecasting pipeline. It houses the legally grounded Environmental Action Advisory Briefs (EAABs) generated by the RAG-LLM agent, alongside the raw semantic evaluation diagnostics used to verify the system's factual accuracy.

By storing these results separately from the execution code, this repository adheres to strict MLOps principles, allowing reviewers to independently audit the model's output without needing to re-run the entire inference pipeline.

---

## 📂 Directory Guide

### 1️⃣ Environmental Action Advisory Briefs (EAABs)
These Markdown files represent the automated, human-readable policy directives generated by the agent across three distinct meteorological scenarios. Every action recommended in these briefs is structurally anchored to the Bangladesh *Air Pollution Control Rules (APCR) 2022*, the *National Air Quality Management Plan (NAQMP) 2024-2030*, and the *WHO 2021 Guidelines*.

*   **`EAAB_Winter_Inversion_Scenario.md`**: Generated under a PURPLE (Crisis) alert triggered by a severe thermodynamic inversion episode. This scenario includes automated "Model Bias Warnings" compensating for tree-based underprediction.
*   **`EAAB_Monsoon_Scenario.md`**: Generated under an AMBER (Alert) stage, highlighting natural wet-scavenging precipitation controls.
*   **`EAAB_Extreme_Scenario.md`**: Generated during a peak extreme pollution event testing the ceiling limits of the National Committee on Air Pollution Control (NCAPC) emergency protocols.

### 2️⃣ Semantic Evaluation Diagnostics
*   **`EAAB_eval_diagnostics.json`**: The raw evaluation log generated by the local RAGAS-Proxy framework.
    *   **Contents:** Contains the unit-by-unit semantic cosine similarities between the generated LLM responses and the retrieved ChromaDB legal context.
    *   **Purpose:** Proves the system's empirical performance, verifying the $0.906$ semantic faithfulness score, the $1.000$ context precision score, and the $0.0\%$ hallucination rate. Thresholds are calibrated to the raw cosine similarity space of the `all-MiniLM-L6-v2` embedding model.
